buildFrom(table) → {string}

Generates the FROM part of query Table can be specified in the following two formats: You should use the string format when the table has no alias: - string: 'user' You should use the array format when the table has a alias(index 0 is table name and index 1 is alias of table)： - array: ['user', 'u']
